# SHAMSU PRD

## Product Goal

SHAMSU is a local AI coding agent inspired by Claude/Cursor-style workflows. It runs locally, uses a local LLM, can read and edit user-provided project files, keeps activity history, supports permission-based tool execution, and protects user code from third-party exposure.

## Core Features

1. Chat interface
2. CLI interface
3. Local LLM through Ollama
4. File reading and editing
5. Ask-before-change permission system
6. Activity history
7. Prompt history
8. Searchable history
9. Login history
10. Context engineering / infinite context handler
11. Tool support
12. MCP support
13. Evaluation harness

## Non-Goals

- Beating Claude in intelligence
- Using cloud APIs by default
- Sending private code to third parties

## Target Users

- Students
- Developers
- Researchers
- Local-first coding assistant users
